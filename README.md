# YouUtility - Pin Player (v1.0.0)

YouUtilityは、YouTubeの動画視聴体験を向上させるChrome拡張機能です。
ページをスクロールして動画プレイヤーが画面外に出ると、自動的に画面の隅にプレイヤーを固定（ピクチャーインピクチャー風）し、コメントや関連動画を見ながら動画視聴を継続できるようにします。

## 主な機能

*   **スクロール連動型固定プレイヤー:**
    *   ページを下にスクロールして動画が見えなくなると、自動的にミニプレイヤー化して画面端に固定されます。
    *   上にスクロールして元の位置に戻ると、自動的に通常の表示に戻ります。
    *   固定化された際も、元のレイアウトが崩れないように設計されています。

*   **カスタマイズ可能な表示:**
    *   **表示位置:** 画面の四隅（左上、右上、左下、右下）から好みの位置を選択できます。
    *   **サイズ変更:** プレイヤーのサイズ（幅）をスライダーまたは数値入力で200px〜1000pxの間で自由に調整できます。

*   **設定の即時反映:**
    *   設定画面での変更は、開いているYouTubeタブにリアルタイムで反映されます。

*   **多言語対応 & テーマ:**
    *   ブラウザの言語設定に合わせて、自動的に日本語または英語に切り替わります（手動設定も可能）。
    *   ライトモード・ダークモード（またはシステム設定準拠）に対応しています。

## インストール方法

**開発者モードでの読み込み:**

1.  このリポジトリをクローンまたはダウンロードし、解凍します。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の「デベロッパーモード」スイッチをオンにします。
4.  「パッケージ化されていない拡張機能を読み込む」をクリックします。
5.  解凍したフォルダ（`manifest.json`が含まれているフォルダ）を選択します。

## 使い方

1.  Chromeに拡張機能をインストール後、YouTubeの動画ページを開きます。
2.  ページを下にスクロールしてコメント欄などを閲覧してみてください。動画プレイヤーが画面端に追従してきます。
3.  設定を変更するには、拡張機能のアイコンをクリックして「オプション」ページを開くか、拡張機能管理画面からオプションを開いてください。

## 設定項目

設定画面 (`options.html`) から以下の項目を変更できます。

### General (一般)
*   **Language (言語):** 日本語と英語から選択できます（初回は自動判定）。
*   **Theme (テーマ):** 設定画面の見た目をライト、ダーク、またはシステム設定に合わせることができます。

### Player Appearance (プレイヤーの表示)
*   **トグルスイッチ:** 固定機能のオン/オフを切り替えます。
*   **Position (位置):** 固定時のプレイヤーの位置を四隅から選択します。
    *   上部（Top）に配置する場合、YouTubeのヘッダーと被らないように自動的に余白が確保されます。
*   **Size (サイズ):** 固定時のプレイヤーの幅を調整します。

## 注意事項

*   本拡張機能はYouTubeの仕様変更により動作しなくなる可能性があります。
*   YouTube公式の「ミニプレーヤー」機能とは異なります。ブラウザ内の要素として動作します。
